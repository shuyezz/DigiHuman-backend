# Digital Human Project
## Description
A project for creating a completed, deployable, user-targeted digital human project. This is the back end part runs under python.

Currently we are using Pytorch as our machine learning framework for pre-trained models supporting, along with Flask as server, expose RESTful APIs to front end.

Currently, this repo is private and allow only collaborators to participate, including submitting issues, forking, modifying source files, etc. We expect opening source for sharing once whole project is finished❤️.
